# 无缝滚动组件

组件地址：Aline/components/Scroll

三方组件：vue3-seamless-scroll
所需参数：

| 参数名称       | 类型               | 说明               |
| -------------- | ------------------ | ------------------ |
| direction      | up,down,left,right | 滚动方向           |
| hover          | Bollean            | 鼠标悬浮是否要停止 |
| limitScrollNum | number             | 最少滚动条数       |
| singleWaitTime | number             | 每步等待时间       |
